# docs/handoffs/

Ephemeral, per-session "you are here" handoffs written by the `creating-handoffs` skill.
The active handoff is `HANDOFF.md`. This is session state, not durable docs — do not put
architecture, follow-ups, or release history here (those belong in ARCHITECTURE.md,
FOLLOWUPS.md/ROADMAP.md, and CHANGELOG.md respectively).
